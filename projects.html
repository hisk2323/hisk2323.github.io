Project 1: Java Epoch Time Converter

This program takes 3 separate inputs from the user: a month (in the form of a digit from 1-12); a day (in the form of a digit from 1-31); and a year (any valid year). It then computes that date in epoch time, and prints this value to the screen. Finally, the program also selects a random date from the same year that the user entered, and also prints this value to the screen. The program takes leap years into consideration to ensure that all printed values are correct. While this project is a little simple compared to what I could code now, I chose to exhibit it on my site because it was really the first big project I did after declaring my Computer Science major, and it feels special to me because of that.

Project 2: History Sniffing via Fake Captcha

This was one of my favorite projects that we did from Information Security. Of course, the design of the actual webpage is pretty bad compared to what we're doing now in Web Development; but the underlying code was pretty interesting. I used CSS to make visited links (a:visited) display the same color as the background, making them virtually invisible. I also made the links unclickable and unselectable, and made it so the mouse wouldn't change on hovering over them. Since the page shows visited links in a different color as unvisited ones, this program exploits that to trick users into entering what they think is a CAPTCHA. However, their input is actually parsed behind the scenes in a PHP server to decipher which of the sites they have visited, and then make a recommendation to them based on that.
